Search

  • Usage
    • ExamplesCode

Search is an input with a button on the right, which displays a result on the screen based on the information entered. The search can happen with each key typed, when pressing enter or when clicking on the button with the magnifying glass, according to the needs.

Search can be used to filter data in lists or tables.

Search field application patterns can be seen on the page of Search Patterns

Positioning‚Äã

By default, the search field should be used on the left, above the list of items and below the Pageheader.

Search interactions‚Äã

With confirmation‚Äã

The search can be performed using the "enter" key or by clicking on the "magnifying glass" button, available in the field itself.

Waiting‚Äã

Allows you to indicate that the system is searching for information.

Field cleaning‚Äã

After typing a term in the search, a button will be displayed with "x" inside the field, to remove the typed term just click on the "x".
